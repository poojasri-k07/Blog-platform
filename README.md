# Blog-platform
Blog platform with user authentication, CRUD blog posts, comments, and MongoDB database integration.

# Blog Platform with Comments

## Features
- User Registration
- User Login & JWT Authentication
- Create Blog Posts
- View Blog Posts
- Update Blog Posts
- Delete Blog Posts
- Add Comments
- View Comments
- MongoDB Database Integration

## Tech Stack
- Node.js
- Express.js
- MongoDB Atlas
- Mongoose
- JWT
- bcryptjs

## Run Locally

npm install
npm run dev
